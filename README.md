# Formula 1-Style Dashboard - Power BI

## Overview
This project is a **Formula 1-style dashboard** built using **Power BI**, designed to provide an interactive and visually appealing report for data analysis. The dashboard includes key performance indicators (KPIs), trend analysis, and insights from the dataset to enhance decision-making and storytelling.

## Features
- **Interactive Visuals:** Dynamic charts and graphs to analyze performance trends.
- **KPIs & Metrics:** Lap times, team performance, driver standings, and race results.
- **Custom Themes:** Formula 1-inspired color schemes for an engaging user experience.
- **Filters & Drilldowns:** Allows users to interact with the data for deeper insights.
- **Data Storytelling:** Clear and concise insights through well-structured reports.

## Dataset
The dashboard is powered by the **AdventureWorks dataset**, which has been processed and modeled within Power BI. The dataset includes:
- Sales performance data
- Customer demographics
- Product analytics
- Regional trends

## Setup Instructions
### Prerequisites
- Install **Power BI Desktop** (latest version) from [Microsoft's official site](https://powerbi.microsoft.com/).
- Download the **AdventureWorks Report_FINAL.pbix** file from this repository.

### Steps to Open and Use
1. Open **Power BI Desktop**.
2. Click on **File > Open** and select `AdventureWorks Report_FINAL.pbix`.
3. Explore the pre-built visuals and interact with the dashboard.
4. Modify and customize as per your requirements.
